# Todo App

### Scope

1. As a user I can add an item to a list of "to-dos"

2. As a user I can see a list of "to-do" items.

3. As a user I can complete an item on the "to-do" list.

### To run locally
nvm use 8.9.3    
npm install  
npm run watch-css  
npm run dub
